Privacy Policy

Last updated: December 18, 2025

Overview

Section titled “Overview”

SpecMon is committed to protecting your privacy. This policy explains what data is collected, how it is used, and your rights regarding your information.

Data collection

Section titled “Data collection”

SpecMon collects minimal data to understand how the website is used and to improve the user experience.

Analytics

Section titled “Analytics”

SpecMon uses Cloudflare Web Analytics, a privacy-first analytics service that:

  • Does not use cookies or track you across websites
  • Does not collect personally identifiable information
  • Complies with GDPR, CCPA, and PECR
  • Provides aggregated, anonymized metrics only

Learn more about Cloudflare’s privacy practices at cloudflare.com/privacypolicy.

Third-party services

Section titled “Third-party services”

Our website links to and interacts with the following external services:

  • GitHub - Source code hosting and community discussions
  • Documentation site (docs.specmon.io) - Technical documentation and guides
  • Academic repositories - Research papers and artifacts (ACM, arXiv, Zenodo)

These services have their own privacy policies and may collect data when you visit them.

Cookies

Section titled “Cookies”

This website does not use cookies. Cloudflare Web Analytics operates without cookies or local storage.

Data retention

Section titled “Data retention”

Analytics data is aggregated and retained according to Cloudflare’s data retention policies. No personally identifiable information is stored.

Your rights

Section titled “Your rights”

Since no personally identifiable information is collected, there is no personal data to access, modify, or delete. If you have concerns about privacy, use the contact information below.

Open source license

Section titled “Open source license”

SpecMon is free and open-source software licensed under the AGPL-3.0 license. The source code is available on GitHub.

Contact information

Section titled “Contact information”

SpecMon is developed and maintained by researchers at the CISPA Helmholtz Center for Information Security.

For questions, issues, or privacy concerns, visit the GitHub Issues page or start a discussion in GitHub Discussions.

Changes to this policy

Section titled “Changes to this policy”

SpecMon may update this privacy policy from time to time. Any changes will be posted on this page with an updated “Last updated” date at the top.
